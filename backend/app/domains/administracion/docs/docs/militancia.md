# Módulo Militancia

Gestión de miembros, agrupaciones territoriales, traslados, skills y disponibilidad.

## Entidades principales

- **Miembro** — identidad, estado, datos de contacto, IBAN
- **TipoMiembro** — catálogo de tipos (militante, simpatizante, colaborador...)
- **AgrupacionTerritorial** — estructura organizativa jerárquica
- **Traslado** — cambio de agrupación de un miembro
- **Skill** — habilidades y nivel de los miembros
- **Disponibilidad** — calendario de disponibilidad para actividades
- **ConsentimientoRGPD** — registro de consentimientos

## Transacciones

Prefijos: `SOC_*`, `SOL_*`, `TRAS_*`, `AGR_*`, `MBR_*`, `SKILL_*`, `AVAIL_*`, `RGPD_*`

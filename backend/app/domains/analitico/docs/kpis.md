# Módulo KPIs

Indicadores clave de rendimiento para actividades y campañas.

## Entidades principales

- **KPI** — indicador genérico con unidad, objetivo y fórmula
- **TipoKPI** — catálogo de tipos
- **KPIActividad** — KPI asociado a una actividad con objetivo y peso
- **MedicionKPI** — medición puntual de un KPI

## Transacciones

Prefijos: `KPI_*`

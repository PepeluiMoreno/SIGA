# Módulo Comunicación Interna

Mensajería interna entre miembros y grupos de la organización.

## Características

- Mensajes directos entre miembros
- Mensajes a grupos de trabajo
- Plantillas de mensaje reutilizables
- Notificaciones automáticas de eventos del sistema

## Transacciones

Prefijos: `MSG_*`, `TPL_*`
